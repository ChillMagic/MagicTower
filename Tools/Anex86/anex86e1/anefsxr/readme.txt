[はじめに]

  　ダウンロード、ありがとうございました。
  Anefsxr は Anex86 の追加モジュールのひとつで、エミュレータ内の MS-DOS から
  パソコンのファイルシステムへのアクセスを提供します。

    このパッケージには以下のファイルが含まれています。
        readme.txt    このファイルです。必ず読んでください。
        anefsxr.e86   Anex86 用の追加モジュールです。
                      anex86.exe と同じディレクトリに置いてください。
                      anex86.exe と同じバージョンを使用してください。
        fsx.com       Anex86 内の MS-DOS 用のプログラムです。
                      実機では絶対に実行しないでください。
                      何が起きても責任持ちません。


[必要なもの]

  　このプログラムを使用するためには同じバージョンの Anex86 が必要です。


[使い方]

  (1). anefsxr.e86 を anex86.exe と同じディレクトリに置きます。
  (2). fsx.com を Anex86 の .fdi か .hdi にコピーしておきます。
  (3). anex86.exe をダブルクリックして Anex86 を起動します。
  (4). [Config]-[FSX] タブを開きます。
  (5). ドライブとパスを指定します。
　　　　ドライブは Anex86 内の MS-DOS のドライブ名です。
　　　　未使用のドライブ名を指定してください。
　　　　必要に応じて MS-DOS の LASTDRIVE を設定してください。
　　　　パスは Anex86 を実行するパソコンのパスです。
  (6). [Start] を押して Anex86 内の MS-DOS を起動します。
  (7). Anex86 内の MS-DOS で fsx.com を実行します。
　　　　必ずメッセージを確認してください。
  (8). 指定したドライブからパスで指定した場所が見えるはずです。


  ※正常に組み込まれたかどうかは、[Config]-[Manager] で確認できます。

  ※Anex86 で使用するアプリケーションとその設定によっては、
    追加モジュールを使うと動くもの、逆に、動かなくなるもの、があります。
    （実機にカードやデバイスを追加したときと同じことです）

  ※最初は追加モジュール無しで試してみることをお勧めします。
    また、問題が生じたときは追加モジュールを外してみてください。


[重要]

  ・このモジュールはパソコンに影響を与える可能性があります。
    何が起きても A.N. は一切責任を持ちません。


[制限事項]

  ・このバージョンでは PC のファイルは読み出ししかできません。
