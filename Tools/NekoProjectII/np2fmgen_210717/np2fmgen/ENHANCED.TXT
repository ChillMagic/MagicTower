INIのENHANCEDまとめ

bit  0 (     1) = メモリD0000-FFFFFの書き込み許可(※)
bit  1 (     2) = タイムスタンパ＆ハードウェアウェイトフラグ(※)
bit  2 (     4) = マルチスレッド(※)
bit  3 (     8) = ステートファイルを上書きする場合元ファイルを.bakにする
bit  4 (    16) = ステートファイルを読み込み前に現在のステートを.s99として保存
bit  5 (    32) = ステートファイルの上書き確認する(※)
bit  6 (    64) = ステートファイルの読み込み確認(※)
bit  7 (   128) = ウィンドウを常に最前面
bit  8 (   256) = シームレスマウス(※)
bit  9 (   512) = オリジナル準拠マウス
bit 10 (  1024) = オリジナル準拠MPU-PC98II
bit 11 (  2048) = IDEフラグ操作(※)
bit 12 (  4096) = Window(Soft)/Stretch Filterを別タイプで描画
bit 13 (  8192) = DirectSoundNotifyを使う(※)
bit 14 ( 16384) = オリジナル準拠PSG
bit 15 ( 32768) = int 18h(ah=45h,46h)をBIOSで実行する(BIOS.ROMがあれば)(※)
bit 16 ( 65536) = 特殊動作モード(※)
bit 17 (131072) = サウンドスピード連動(※)
bit 18 (262144) = マウスチャタリング防止(※)
bit 19 (524288) = IDE/SCSIマウントイメージ名表示(※)

使用したい機能のビットを2進数とするかまたは(10進数)内を合計して設定します
2進数記述方法は(INI.txt参照)
